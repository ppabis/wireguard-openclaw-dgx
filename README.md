OpenClaw -> Wireguard -> Ollama -> DGX Spark
=========

An example infrastructure to running OpenClaw or any other agent on EC2 instance
while using a local home machine to use as the inference backend. You can use
this setup to host an open-weights model or for any skills like generating
images with Z-Image-Turbo, videos with LTX-2.3, ASR or TTS.
